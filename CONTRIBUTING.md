# Contributing (Frontend)

## Setup
- Clone this repo
- Start a local server: `python -m http.server 8080`

## Branching & Commits
- Branch from `main`: `feat/your-change`, `fix/your-bug`
- Conventional commits: `feat:`, `fix:`, `docs:`, `chore:`

## API URL
- Edit `main.js` `apiUrl` for local vs production

## PRs
- Keep changes focused; describe what/why in the PR
